Documentación de la API de Apk Mods

¡Hola! Si estás aquí, probablemente quieres integrar los datos de Apk Mods en tu propio proyecto. ¡Genial! Hemos creado esta API para que puedas acceder a nuestra colección de mods de forma sencilla y programática. Aquí te explicamos cómo usarla, sin rodeos y con ejemplos claros.

URL Base de la API

https://apk-mods-one.vercel.app/api/mods

Nota importante: Esta API es de solo lectura (GET). No puedes modificar ni añadir mods a través de ella. Para añadir mods, usa nuestra interfaz web.

Endpoints Disponibles

GET /api/mods

Obtiene una lista de todos los mods disponibles, o filtra por criterios específicos.

Parámetros de Consulta (Query Parameters)

Parámetro Tipo Descripción Ejemplo
id number Filtra por el ID único del mod. ?id=123
name string Busca mods cuyo nombre contenga el texto especificado (búsqueda parcial, insensible a mayúsculas/minúsculas). ?name=whatsapp
description string Busca mods cuya descripción contenga el texto especificado (búsqueda parcial, insensible a mayúsculas/minúsculas). ?description=grabador
version string Busca mods por su versión (búsqueda parcial, insensible a mayúsculas/minúsculas). ?version=2.23
developer string Busca mods por el nombre del desarrollador (búsqueda parcial, insensible a mayúsculas/minúsculas). ?developer=moddev
download_link string Busca mods por su link de descarga (búsqueda parcial, insensible a mayúsculas/minúsculas). ?download_link=mediafire
search string Realiza una búsqueda de texto completo inteligente en los campos de nombre, descripción y desarrollador. Ideal para búsquedas generales. ?search=grabador de pantalla

Ejemplos de Uso

Obtener todos los mods
curl https://apk-mods-one.vercel.app/api/mods
fetch('https://apk-mods-one.vercel.app/api/mods')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));
Buscar mods por nombre
curl https://apk-mods-one.vercel.app/api/mods?name=whatsapp
fetch('https://apk-mods-one.vercel.app/api/mods?name=whatsapp')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));
Búsqueda inteligente (texto completo)
curl https://apk-mods-one.vercel.app/api/mods?search=grabador de pantalla
fetch('https://apk-mods-one.vercel.app/api/mods?search=grabador de pantalla')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

Formato de la Respuesta

La API siempre devolverá un array de objetos JSON. Si no se encuentran mods, devolverá un array vacío.

[
  {
    "id": 1,
    "name": "WhatsApp Plus",
    "description": "Versión modificada de WhatsApp con funciones extra.",
    "version": "v19.50.0",
    "developer": "AlexMods",
    "download_link": "https://ejemplo.com/whatsapp-plus",
    "icon_url": "https://i.ibb.co/your-image.png"
  },
  {
    "id": 2,
    "name": "XRecorder Pro",
    "description": "Grabador de pantalla avanzado para Android.",
    "version": "v1.2.3",
    "developer": "RecorderDev",
    "download_link": "https://ejemplo.com/xrecorder-pro",
    "icon_url": "https://i.ibb.co/another-image.png"
  }
]

Errores

En caso de un error en el servidor, la API devolverá un código de estado HTTP 500 Internal Server Error con un objeto JSON que contendrá un mensaje de error.

{
  "error": "Internal Server Error"
}
